About

WhatsApp chatbot for querying maritime vessel PSC (Port State Control) data using Twilio, Claude AI, and Netlify serverless functions.

Available Functions

WhatsApp Webhook:

/.netlify/functions/whatsapp-webhook

Generate Excel:

/.netlify/functions/generate-excel

Download Excel:

/.netlify/functions/download-excel

Send Email:

/.netlify/functions/send-email

Usage

This bot is accessed via WhatsApp through Twilio. Send messages to your configured Twilio WhatsApp number to interact with the bot.

Example queries:

  • "Risk score for GCL YAMUNA"
  • "Risk level of GCL TAPI"
  • "Recommendations for GCL GANGA"
